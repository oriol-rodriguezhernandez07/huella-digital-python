# huella-digital-python
17. Huella digital de Audio y el Algoritmo de Shazam. Programación 1ºGIA UPM
Componentes del grupo:

Proyecto académico para la extracción de huellas digitales de audio utilizando STFT, detección de picos y hashes no deterministas.

## Requisitos
- Python 3.x
- essentia
- numpy
- matplotlib
- scipy
- datetime

**Nótese que la librería essentia sólo es compatible para sistemas operativos basados en Linux. En caso de utilizar WSL en Windows, asegúrese de ejecutar el código desde la carpeta de instalación del subsistema

Para que el programa funcione satisfactoriamente, debe sustituir manualmente en el código la ruta del archivo de audio que desea cargar y dónde guardar los gráficos generados. En el código ya se encuentran estas rutas indicadas pero a modo de ejemplo.
